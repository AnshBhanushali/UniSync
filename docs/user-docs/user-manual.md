# User Manual – Using UniSync

This user manual provides step-by-step instructions for interacting with UniSync and understanding its responses.

## Asking a Question
To ask a question:
1. Enter your question in the text input field.
2. Use clear and complete sentences when possible.
3. Submit the question and wait for the response.

Examples:
- “How do I apply as an international student?”
- “What is the deadline for graduate applications?”

## Understanding UniSync’s Responses
UniSync generates answers using an AI model combined with information retrieved from public University of Cincinnati web pages.

Responses may include:
- A summarized answer
- Contextual explanations
- References to relevant university webpages

The answer is intended to guide you, not serve as an official policy statement.

## Limitations
UniSync has the following limitations:
- It only uses **publicly available UC information**
- It may not always reflect the most recent updates
- It cannot access personal student records
- It may not answer highly specific or uncommon questions

If an answer seems incomplete or unclear, users should consult official university resources.

## Responsible Use
UniSync should be used responsibly:
- Do not rely on UniSync for critical academic or legal decisions
- Always verify important information with official UC sources
- Understand that UniSync is a student-built project

## Error Handling
If UniSync cannot answer a question:
- Try rephrasing the question
- Ask a more general version of the question
- Refer directly to the University of Cincinnati website

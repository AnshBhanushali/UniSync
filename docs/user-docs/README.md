# UniSync – User Documentation

UniSync is an AI-powered assistant designed to help University of Cincinnati students quickly find answers to common academic and administrative questions using publicly available university information.

This documentation provides a guide for users to understand what UniSync does, how to use it, and what its limitations are.

## Who is this for?
- Current University of Cincinnati students
- Prospective students seeking general information
- Users looking for quick answers from public UC websites
